# うん言語仕様表

## 文

- `うんちく <text>`: コメント（実行時は無視）
- `うん <name> = <expr>`: 変数定義
- `うーん <expr>`: 式を評価（結果は捨てる）
- `うん！ [<expr>]`: 終了。expr 省略時は直前の値を返す

## 演算子

優先順位: `うんかけ` / `うんわり` → `うんたす` / `うんひく`

- `うんたす`: 足し算 (+)
- `うんひく`: 引き算 (-)
- `うんかけ`: 掛け算 (\*)
- `うんわり`: 割り算 (/)。右辺が 0 なら runtime error

## リテラル・変数

- 数値: `100`, `-3`, `3.14`
- `うんなし`: 0 / false
- `うんあり`: 1 / true
- 変数名: 英字始まりの英数字（例: `x`, `total`）
- `( <expr> )`: 括弧で優先順位を変更

## エラー表示

すべての runtime error は **`うんともすんとも`** のみ表示される（行番号・原因は出ない）。

## CLI

```txt
unlang run <file>    # 実行し、戻り値を stdout に出力
unlang check <file>  # 構文・実行チェック（成功時 ok）
```

## 例

```txt
うんちく 売上集計
うん x = 100
うん y = うんあり
うん z = x うんわり y
うん！ z
```

上記は `100 / 1 = 100` を返す。`y` が `うんなし` だと 0 除算で `うんともすんとも` になる。
