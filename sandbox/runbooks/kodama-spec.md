# こだま仕様表

## 文

- `やまびこ帳 <text>`: コメント（実行時は無視）
- `よぶ <name> = <expr>`: 変数定義
- `はかる <expr>`: 式を評価（結果は捨てる）
- `もし？ <cond>`: 条件（仕様上のみ。現行インタープリタは未実装）
- `かえす [<expr>]`: 終了。expr 省略時は直前の値を返す

## 演算子

優先順位: `かける` / `わる` → `たす` / `ひく`

- `たす`: 足し算 (+)
- `ひく`: 引き算 (-)
- `かける`: 掛け算 (\*)
- `わる`: 割り算 (/)。右辺が 0 なら runtime error

## リテラル・変数

- 数値: `100`, `-3`, `3.14`
- `しずか`: 0 / false
- `こだま`: 1 / true
- 変数名: 英字始まりの英数字（例: `x`, `total`）
- `( <expr> )`: 括弧で優先順位を変更

## エラー表示

すべての runtime error は **`こだまが返ってきません`** のみ表示される（行番号・原因は出ない）。

## CLI

```txt
kodama run <file>    # 実行し、戻り値を stdout に出力
kodama check <file>  # 構文・実行チェック（成功時 ok）
```

## 例

```txt
やまびこ帳 売上集計
よぶ x = 100
よぶ y = こだま
よぶ z = x わる y
かえす z
```

上記は `100 / 1 = 100` を返す。`y` が `しずか` だと 0 除算で `こだまが返ってきません` になる。
